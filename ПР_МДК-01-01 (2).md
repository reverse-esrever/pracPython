## 1.1 Переменные и вывод на экран

### Теория

**Переменная** — это именованное место в памяти, где хранятся данные. В Python переменные создаются очень просто: достаточно написать имя и присвоить значение.

Для вывода информации используется функция `print()`.

### Пример 1 - Создание переменной

```python
name = "Иван"
age = 20

print("Имя:", name)
print("Возраст:", age)
```

**Вывод:**
```
Имя: Иван
Возраст: 20
```

## 1.2 Типы данных

### Теория

В Python есть основные типы данных:
- **int** — целые числа (10, -5, 0)
- **float** — числа с плавающей точкой (3.14, -0.5)
- **str** — строки ("Hello", 'Python')
- **bool** — логические значения (True, False)

### Пример 2 - Разные типы данных

```python
health = 100          # int
speed = 5.5           # float
player_name = "Герой" # str
is_alive = True       # bool

print("Здоровье:", health)
print("Скорость:", speed)
print("Имя:", player_name)
print("Живой:", is_alive)
```

**Вывод:**
```
Здоровье: 100
Скорость: 5.5
Имя: Герой
Живой: True
```

## 1.4 Умножение и деление

### Теория

- `*` — умножение
- `/` — обычное деление (всегда возвращает float)
- `//` — целочисленное деление

```python
price = 50
quantity = 3

print("Каждому по:", each)

```

## 1.5 Целочисленное деление и остаток

### Теория

Целочисленное деление `//` возвращает только целую часть результата.
Оператор `%` возвращает остаток от деления.

### Пример 5 - Целочисленное деление

```python
apples = 17
baskets = 5

per_basket = apples // baskets
remaining = apples % baskets

print("Яблок всего:", apples)
print("Корзин:", baskets)
print("В каждой корзине:", per_basket)
print("Остаток:", remaining)
```

**Вывод:**
```
Яблок всего: 17
Корзин: 5
В каждой корзине: 3
Остаток: 2
```

## 1.7 Приоритет операций

### Теория

Операции выполняются в определенном порядке:
1. Скобки `()`
2. Степень `**`
3. Умножение, деление `*`, `/`, `//`, `%`
4. Сложение, вычитание `+`, `-`

### Пример 7 - Приоритет

```python
result1 = 10 + 5 * 2
result2 = (10 + 5) * 2

print("10 + 5 * 2 =", result1)
print("(10 + 5) * 2 =", result2)

# Сложный пример
result3 = 2 + 3 * 4 - 5
result4 = (2 + 3) * (4 - 5)

print("\n2 + 3 * 4 - 5 =", result3)
print("(2 + 3) * (4 - 5) =", result4)
```

**Вывод:**
```
10 + 5 * 2 = 20
(10 + 5) * 2 = 30

2 + 3 * 4 - 5 = 9
(2 + 3) * (4 - 5) = -5
```

## 2.1 Оператор if

### Теория

Условный оператор `if` позволяет выполнять код только при выполнении определенного условия.

Операторы сравнения:
- `==` — равно
- `!=` — не равно
- `>` — больше
- `<` — меньше
- `>=` — больше или равно
- `<=` — меньше или равно

### Пример 1 - Простое условие

```python
health = 30

if health < 50:
    print("ВНИМАНИЕ! Низкое здоровье!")
    print("Используйте зелье")
```

**Вывод:**
```
ВНИМАНИЕ! Низкое здоровье!
Используйте зелье
```

## 4.2 Доступ к элементам списка

### Теория

Элементы списка нумеруются с 0. Доступ к элементу — через индекс в квадратных скобках.

```python
fruits = ["Яблоко", "Банан", "Апельсин", "Груша"]

print("Первый фрукт:", fruits[0])
```

```python
    if gold >= price:
        inventory.append(item_name)
        gold = gold - price
        print(f"Куплено: {item_name}")
        return gold
    else:
        print("Недостаточно золота!")
        return gold

# Основная программа

player_gold = buy_item(player_inventory, "Зелье", 50, player_gold)
player_gold = buy_item(player_inventory, "Меч", 200, player_gold)
```
